# DNS配置最简单方案 - 操作指南

## 📌 方案概述
只使用 **www.pst56.com** 作为网站域名，保留根域名 **pst56.com** 的所有DNS记录（包括邮箱）不变。

✅ **优点：**
- 操作最简单，只需添加1条DNS记录
- 邮箱完全不受影响
- 风险最低，不会破坏现有配置
- 网站正常显示

---

## 🚀 操作步骤（只需3步）

### 第1步：在Netlify上添加域名

1. 登录 Netlify
2. 进入您的项目 **pst56**
3. 点击左侧菜单 **Domain management**（域名管理）
4. 点击 **Add domain alias**（添加域名别名）
5. 输入：`www.pst56.com`
6. 点击 **Verify**（验证）
7. 等待Netlify显示DNS配置信息

---

### 第2步：在您的域名管理后台添加DNS记录

**在哪里操作？**
- 如果域名在阿里云买的 → 登录阿里云 → 域名控制台 → DNS解析
- 如果域名在腾讯云买的 → 登录腾讯云 → 域名解析
- 如果域名在其他平台 → 找到"DNS解析"或"域名解析"功能

**具体操作：**
1. 找到域名 `pst56.com` 的DNS解析设置
2. 点击 **添加记录**
3. 填写以下信息：
   ```
   记录类型：选择 "CNAME" 或 "CNAME记录"
   主机记录：填写 "www"
   记录值：填写 Netlify 提供的值（通常是 "pst56.netlify.app"）
   TTL：选择默认值或600
   ```
4. 点击 **确定** 或 **保存**

⚠️ **重要提示：**
- **不要删除任何现有的DNS记录**（特别是MX记录和TXT记录）
- **只添加这一条新的CNAME记录**
- 主机记录填 `www`，不要填 `@` 或其他

---

### 第3步：等待DNS生效

1. DNS记录生效通常需要 **10分钟到24小时**
2. 可以在Netlify的Domain management页面查看状态
3. 看到绿色勾号 ✓ 表示配置成功

---

## ✅ 完成后如何访问网站

- ✅ **www.pst56.com** → 显示您制作的网站
- ✅ **邮箱** → 继续正常使用（www.pst56.com 的邮箱）
- ⚠️ **pst56.com**（没有www）→ 可能还是显示旧网站，这是正常的，不影响

---

## 🆘 如果遇到问题

### 问题1：DNS记录添加后不生效
- 等待更长时间（最多24小时）
- 清除浏览器缓存
- 用无痕模式访问测试

### 问题2：Netlify显示验证失败
- 检查DNS记录是否正确添加
- 确认主机记录填写的是 `www`（不是 `@`）
- 确认记录值填写正确

### 问题3：邮箱不能用了
- 立即检查DNS解析中是否误删了MX记录
- 如果删除了，联系邮箱服务商恢复MX记录
